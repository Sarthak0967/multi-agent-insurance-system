# Multi-Agent Insurance Research System

A multi-agent system built using CrewAI that:
- Researches insurance policies
- Structures information
- Demonstrates policy details

## Tech Stack
- Python
- CrewAI
- Google Gemini
- Web Scraping Tools

## Agents
1. Research Agent
2. Demonstrator Agent

## How to Run

1. Clone repo
2. Create virtual environment
3. Install dependencies:
   pip install -r requirements.txt
4. Add your API key in .env:
   GOOGLE_API_KEY=your_key_here
5. Run:
   python main.py
